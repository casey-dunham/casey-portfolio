# Portfolio

**Personal portfolio site at [caseydunham.design](https://caseydunham.design).**

Built with Next.js, GSAP, and Framer Motion. Features smooth scroll-driven animations, responsive layout, and a project showcase with lightbox galleries.

## Tech Stack

- Next.js / TypeScript
- GSAP + Framer Motion for animations
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```
